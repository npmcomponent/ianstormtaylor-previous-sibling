# previous-sibling

  Get the previous sibling of an `el`, optionally filtered by a `selector`.

## Installation

    $ component install ianstormtaylor/previous-sibling

## Example
  
```js
var previous = require('previous-sibling');
var head = previous(document.body); // document.head
```
